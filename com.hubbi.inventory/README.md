# Hubbi Inventory & WMS

Módulo de gestión de inventario universal para Hubbi.

## Características

- **Vista de Tabla**: TanStack Table con búsqueda, ordenamiento y exportación a Excel.
- **Vista WMS (3D)**: Visualización isométrica de bodegas con Three.js.
- **Universal**: Soporta productos, servicios, activos fijos y kits.
- **Multi-UOM**: Unidades de medida múltiples por producto.
- **Campos Dinámicos**: Define atributos personalizados globales.

## Permisos

| Permiso | Descripción |
|---------|-------------|
| `inventory.access` | Ver el módulo |
| `inventory.create` | Crear productos |
| `inventory.update` | Editar productos |
| `inventory.delete` | Eliminar productos |
| `inventory.adjust_stock` | Ajustar inventario |
| `inventory.map_edit` | Editar mapa de bodega |
| `inventory.approve_transfer` | Aprobar transferencias |
| `inventory.export` | Exportar datos |

## Instalación

1. Descarga el archivo `.hubbi`
2. Abre Hubbi → Tienda de Módulos
3. Importa el archivo
