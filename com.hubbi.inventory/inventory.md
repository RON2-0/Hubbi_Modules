Módulo de Inventario

Este módulo debe ser un inventario robusto, multifacético y modular, tanto para pequeños inventarios como para grandes. Para empresas enormes, fábricas, hoteles, domótica, talleres, bibliotecas, librerías, cosméticas, restaurantes, etc.

Funcionalidades: 

- Productos: Dos vistas, una como tabla/Excel y una como VMS

- Tipos de productos, para distintos tipos de negocios, es decir debe haber una opción en los ajustes para poder detenerminar el tipo de administración de productos o inventario, así como agregar campos personalizados.
- Fecha de vencimiento debe poder activarse o desactivar, desactivado por defecto.
- Elementos fijos y no fijos Inventarios de venta
- Inventario de uso interno 
- En el futuro este módulo deberá ser compatible para trabajar en conjunto con un módulo de facturación, manteniendo y otros.

El dashboard debe de poder ser personalizable, es decir, según el uso qeu se le de o el negocio se deben de poder poner diferentes widgets para personalizar el tablero. Para analálisis de información, graficas, estadísticas, o acceso a funcionalidades rapidas.

- Posibilidad de tener cantidades enteras y fracciones, unidades, precio de compra y precio de venta. Se deberá poder leer la información de los proveedores del modulo futuro de compras, stock minimo y maximo, etc.
